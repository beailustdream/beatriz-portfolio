# 📝 To-Do List

Aplicação de lista de tarefas desenvolvida com **HTML, CSS e JavaScript**, com foco em manipulação de DOM, lógica de programação e persistência de dados.

---

## ✨ Funcionalidades

* ➕ Adicionar tarefas
* ✅ Marcar tarefas como concluídas
* ❌ Remover tarefas
* 🔍 Filtrar tarefas (todas, ativas, concluídas)
* 💾 Persistência de dados com localStorage
* ⌨️ Adicionar tarefa pressionando Enter

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

---

## 📂 Estrutura do projeto

```bash
todo-list/
│
├── index.html
├── style.css
└── script.js
```

---

## 💻 Como executar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/SEU-USUARIO/todo-list-js.git
```

2. Acesse a pasta do projeto:

```bash
cd todo-list-js
```

3. Abra o arquivo `index.html` no navegador
